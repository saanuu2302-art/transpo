# **App Name**: Transpo

## Core Features:

- Secure User Authentication: Implement secure login/sign-up using phone OTP or email/password for farmers, machine owners, and drivers.
- Vehicle Booking: Allow farmers to book vehicles (Tractor, Mini-Truck, etc.) by viewing available options with images, cost, and driver ratings. Includes real-time tracking.
- Machine Booking: Enable farmers to book agricultural machinery (tractor, tiller, harvester, etc.) View availability, rental cost, and machine owner details.
- AI Expert Assistance: Integrate an AI-powered module where farmers can input queries via text or voice and receive responses in text or voice format. The AI tool will intelligently consider various data points to offer farming advice and solutions.
- Machine Owner Management: Machine owners can upload machines with details (name, image, cost/hr, location), manage availability, view farmer rental requests, and manage bookings.
- Driver/Transport Provider Interface: Provide drivers with a secure login, a list of incoming farmer ride requests, navigation via maps, trip start/stop functionality, and a wallet earnings page.
- Bilingual Support: Every screen displays text in both Kannada and English.

## Style Guidelines:

- Primary color: Earthy Green (#6B8E23), evoking growth and nature, reflecting the agricultural theme.
- Background color: Light beige (#F5F5DC), a desaturated hue that provides a neutral backdrop for the green.
- Accent color: Golden Yellow (#FFD700) used to highlight calls to action and important information, representing harvest and prosperity.
- Headline font: 'Playfair', a modern serif, elegant and readable in both English and Kannada
- Body font: 'PT Sans', a clear, accessible sans-serif, suitable for labels and text blocks; pairs well with Playfair.
- Use vector icons and illustrated farming art, with big, friendly icons for low-literacy users, representing different vehicles, machinery, and farming tasks.
- Employ neo-glassmorphism or neo-material design, with a modern, clean, and intuitive layout. Use blurred, high-quality farming backgrounds.
- Implement smooth animated transitions throughout the application to enhance user experience.
